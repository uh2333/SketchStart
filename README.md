## 🚀 普通用户快速开始

如果你是 **普通用户**，只想用工具画图，请直接下载 [Releases](https://github.com/uh2333/SketchStart/releases) 中的压缩包：

1. 解压文件。
2. 双击 `SketchTool.exe` 启动后台。（**exe 文件不可移动！** 如果需要放桌面请使用快捷方式）
3. 浏览器会自动打开工具页面（地址 `http://127.0.0.1:8443/static/index.html`）。
4. **关闭后台窗口 = 关闭服务**，网页将无法访问。

> 💡 后台开启后，你可以把它最小化。之后每次使用只需在浏览器打开网站即可，无需再次双击 exe（前提是后台未关闭）。  
> 压缩包中自带了一个网页入口和 exe 快捷方式。

## 💻 从源码运行

如果你想自己运行源码、修改或二次开发，请按照以下步骤操作。

### 环境要求
- Python 3.10+
- pip 或 conda

### 安装依赖

**使用 pip：**
```bash
pip install -r requirements.txt
```

**使用 conda：**
```bash
conda env create -f environment.yml
conda activate sketch_env
```

### 启动服务
```bash
python main.py
```

启动后，浏览器会自动打开 `http://127.0.0.1:8443/static/index.html`。

## 🔌 插件

项目支持插件扩展，目前内置了以下插件：

| 插件 | 功能 |
| :--- | :--- |
| ☁️ 百度云盘 | 基于 bypy，首次使用需授权。可读取百度云 `/apps/bypy` 下的 `.zip` 图包和图片。 |
| 🎨 色彩调节 | 调整图片黑白 / 色彩视图。 |
| 🔄 图片变换 | 旋转 / 翻转当前图片。 |

### 自制插件

**非常欢迎**你为这个工具写插件！

将插件文件夹放入 `data/plugins/` 目录，重启后台即可自动加载。  
需要使用 `manifest.json` 注册插件（具体可以参考其他插件的对应文件）。

> ⚠️ **关于依赖的说明**：
> - 如果你是**从源码运行**，插件需要的额外 Python 包可以自行 `pip install` 安装。
> - 如果你使用的是 **Releases 中的 exe 版本**，由于 exe 是预先打包好的，**插件如果需要额外的依赖包，目前无法自动加载**，需要重新打包整个 exe 才能生效。
> - 建议插件开发者尽量使用 `environment.yml` 中已有的依赖，这样插件对 exe 用户也能即插即用。

> 📌 **插件开发说明**：
> 插件的相关 API 和开发文档我还在整理中，后续会补充完善。  
> 目前如果需要开发插件，建议先参考已有插件的实现方式（如 `data/plugins/` 下的示例）。

如果你写了好用的插件，也欢迎提交 Pull Request 分享给更多人～

## 🌐 局域网访问

支持在同一 Wi-Fi 下用手机 / 平板 / 其他电脑访问。

- 在软件设置中开启「局域网访问」。
- 首次开启可能需要 **防火墙放行**。建议选择「专用网络」，并确保当前 Wi-Fi 在 Windows 中被设为“专用网络”。
- 每次启动会记忆上一次的局域网状态。

开启后，手机扫码或直接访问控制台显示的 IP 地址即可。

## 📂 文件存储位置

| 类型 | 路径 | 说明 |
| :--- | :--- | :--- |
| 图包 | `data/sketch_packages/` | 仅支持 `.zip` 格式，可以直接在本地管理 |
| 收藏图片 | `data/favorites/` | 按分类保存为图片文件 |
| 练习进度 | `data/metadata/` | 记录每张图片的完成状态 |
| 插件 | `data/plugins/` | 自制插件放在这里 |
